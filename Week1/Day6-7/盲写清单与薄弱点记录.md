## 二、常见薄弱点（容易卡住/写错的地方）

| 易错点 | 正确做法 | 说明 |
|--------|----------|------|
| stride 单位 | 用**字节**，不是 float 个数 | `stride = 5 * 4 = 20` 字节，不是 5 |
| offset 单位 | 也用**字节** | 颜色 offset = `2 * 4 = 8`（跳过 x,y） |
| 忘记 enableVertexAttribArray | 每个 attribute 都要 enable | 不 enable 该 attribute 读不到数据 |
| 忘记 useProgram | draw 前必须 `useProgram` | 否则用的不是你以为的 program |
| uniform 在 draw 前设置 | 每帧 `uniform1f` 再 draw | uniform 是「当前 program」的状态 |
| 片元缺少 precision | `precision mediump float;` | 片元着色器 float 必须声明精度 |
| getUniformLocation 判空 | 用 `=== null` 判断 | 注意它返回 null，不是 -1 |
| getAttribLocation 判空 | 用 `< 0` 判断 | attribute 找不到返回 -1 |
| 颜色没变化/纯色 | 确认 varying 名字 VS/FS 完全一致 | varying 靠**同名**连接 VS 和 FS |
| ES Module 报错 | `<script type="module">` + 用 http 打开 | `file://` 直接打开会被 CORS 拦 import |
